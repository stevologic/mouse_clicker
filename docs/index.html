Everything a clicker should be

Extremely configurable

Dial in exactly the behavior you need — from a blistering 120 CPS to a slow, human-looking idle jiggle.

⟳

Every click type

Left, right, middle, or side buttons. Single, double, triple, N-click bursts, scroll, or press-and-hold — with randomized hold times that read as human.

⏱

Precise timing

Fixed or randomized intervals, quick CPS presets, a start countdown, and run limits by click count or duration.

↗

Anywhere on screen

Click at the cursor, a fixed point, a random spot inside a region, or walk a looping sequence of points across every monitor.

〰

Humanized movement

Teleport, glide, or move along curved Bézier paths with easing and micro-jitter so the cursor looks hand-driven, not scripted.

⌨

Global hotkeys

Start, stop, and emergency-stop from anywhere with system-wide hotkeys — even while ClickForge sits in the background.

✦

AI patterns

Describe what you want in plain English and let Claude build the whole pattern. Works offline too, with a built-in generator and presets.

✧

Living interface

A hand-built animated UI — a cursor-reactive particle constellation, an aurora backdrop, glassmorphism, and live click pulses. It idles to near-zero CPU when in the background.

Powered by Claude

Just describe it

Type what you want in plain English. ClickForge asks Claude (with your own Anthropic API key) to translate it into a precise, ready-to-run pattern.

"Click like a human every 1–3 seconds near the center of the screen, with slight random movement, for 5 minutes."

No key? A built-in offline generator and one-click presets — rapid fire, human idle jiggle, gentle clicks, region spray — have you covered. Your key is stored locally and sent only to Anthropic.

Get started

Running in 30 seconds

Nothing to install. Download the executable and run it — or build it yourself from source.

Download

Grab ClickForge.exe from the latest release.

Run it

Double-click. It runs on the .NET Framework already built into Windows — no runtime download.

Configure or ask AI

Set your options, or open the AI tab and describe the pattern you want.

Press your hotkey

Hit F6 to start and stop from anywhere. F8 is the emergency stop.

Prefer to build from source? You only need Windows — the compiler ships with it.

git clone https://github.com/stevologic/mouse_clicker
cd mouse_clicker
powershell -ExecutionPolicy Bypass -File build.ps1 -Run
